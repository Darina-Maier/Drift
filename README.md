Template pour un jeu phaser disposant d'un niveau selection et de 3 niveaux.
Le tmeplate possèdé également la structure pour rassembler des fonctions dans un meme fichier et les importer dans d'autres fichiers


Nom du jeu : DRIFT 
Membres du groupe : MAIER Darina (Cheffe de projet) ; TOURNAMILLE Charlotte ; KASSAB Zeinab 

Commandes du jeu : 
Flèche Droite : Avancer à droite 
Flèche Gauche : Avancer à gauche 
Espace : Sauter
G :Inverser la gravité (dans certains niveaux)
Flèche du haut : Retourner au menu principal 
A : Tirer (disponible dans certains niveaux) 
Clique souris : Interagir avec les menus/boutons 

Description rapide du jeu :
Notre concept: DRIFT est un jeu de plateforme 2D avec un concept unique de gravité inversible. Le joueur incarne un astronaute qui doit explorer différentes planètes tout en maîtrisant le contrôle de la gravité pour progresser et atteindre son objectif.

Objectifs du joueur :
- Traverser les 7 planètes remplis de défis et d'ennemis
- Collecter les pièces de la fusée pour progresser et arriver sur Terre.
- Utiliser la mécanique de gravité inversible pour pouvoir réussir les défis de chacune des planètes. 
- Atteindre le téléporteur à la fin de chaque niveau pour avancer pour rentrer sur la Terre.
